# Smart Quiz Pro

Smart Quiz Pro is a modern interactive quiz web application built using HTML, CSS, and JavaScript.

## Features
- Interactive quiz interface
- Timer-based quiz system
- Score tracking
- Instant answer feedback
- Responsive design
- Multiple question handling
- Smooth UI animations

## Technologies Used
- HTML5
- CSS3
- JavaScript
